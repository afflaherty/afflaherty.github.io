Poetic Variation - Master's Project

My graduate project was analysis of poetry. I used Mathematica and R code to perform factor analysis on various works of poetry to determine if patterns existed for different authors based on parts of speech. The analysis resulted in five factors that contributed towards the works analysed. The first factor contained positive loadings for nouns and prepositions and a negative loading for pronouns, demonstrating that the more nouns present, which are usually preceded by a preposition, implies the diminished use of pronouns. The full explanation of each factor, along with the full code use, can be found in the full write-up.
