Analysis

Master's Project

My graduate project was analysis of poetry. I used Mathematica and R code to perform the analysis on a text which separated parts of speech.

Example of Mathematica code

adjective = StringSplit[ToString[Import[ "\Users\owner\Documents\MATLAB\MASTERS\Parts of Speech\adjective.txt"]], \ {Whitespace}];
AdjectiveList = Flatten[Table[Intersection[{stringlist[[i]]}, adjective], {i, Length[stringlist]}]] Length[AdjectiveList]
